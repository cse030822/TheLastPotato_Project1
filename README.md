# 화성 정원사 · Terra-Garden 🔴🥔

손으로 **총 쏘는 동작**을 하면 화성 배경 속 테라-건이 조준·발사 상태를 인식하는 게임.
현재 범위: **0단계 ~ 8단계**. 씨앗 심기 → 에너지로 감자를 키워 **감자알을 수확**하고, 몰려오는 곤충으로부터 밭을 지켜 승리하는 한 판이 완성돼 있다.

## 실행

```bash
pnpm install
pnpm dev        # http://localhost:5173 (또는 표시되는 주소)
```

실행하면 **인트로(타이틀) 화면**이 뜬다. 제목 아래 **`START`** 버튼을 누르면 **카메라 권한 안내 화면**으로 넘어가고, 거기서 **`카메라 켜고 시작하기`**를 누르면 그때 **웹캠 권한**을 묻는다(권한을 허용해야 손 인식이 동작). 로드가 끝나면 게임이 시작된다. 권한을 거부하면 안내가 뜨고 다시 시도할 수 있다. 게임 도중에는 **`[H]`** 키나 상단 **`❔ 도움말`** 버튼으로 플레이 방법 창을 언제든 열어 볼 수 있다(열려 있는 동안 게임은 잠시 멈춘다).

```bash
pnpm build      # 타입체크(tsc) + 프로덕션 번들
pnpm preview    # 빌드 결과 미리보기
```

## 조작

- 손을 웹캠에 비추면 우하단 PIP 창에 잡히고, 상단 HUD에 **왼손/오른손 상태**가 표시된다.
- **손가락 총 자세**(검지 펴기 + 엄지 세우기 + 나머지 접기) → `GUN POSE`(조준), 노즐 끝이 약하게 발광.
- 그 상태에서 **검지를 뿌리 쪽으로 당기면** → `FIRE`(발사), 노즐 끝이 강하게 발광.
- 손을 좌우/상하로 움직이면 해당 쪽 테라-건 **노즐이 따라 회전**한다. 손이 사라지면 정면으로 복귀.
- **[D]** 키: 손 뼈대(관절 점 + 반투명 흰 선) 오버레이 켜기/끄기.
- **[Space]** 키: (씨앗 단계) 제스처가 잘 안 될 때 씨앗 심기 보조. **[R]** 키: 승패 후 다시 시작.

> 오른손 = 형광 블루(생명 에너지) 테라-건, 왼손 = 붉은 불꽃(방어) 테라-건.
> 거울 모드에서도 **화면상 오른쪽에 보이는 손 = 오른손**으로 확정한다(화면 x 위치 기준).

## 폴더 구조 (역할별 분리)

```
src/
├─ mars/        화성 3D 공간: 씬/카메라/조명/바닥/하늘, 테라-건, 장식(감자·곤충)
├─ hands/       손 인식·제스처: 웹캠+MediaPipe, 뼈대 오버레이, GUN POSE/FIRE 판별
├─ particles/   물·불꽃 파티클 (5단계 예정 — 현재 비어 있음)
├─ game/        게임 진행·HUD
└─ main.ts      전체 조립 + 렌더 루프
```

### 성장·수확(6~8단계) 흐름

- 오른손 에너지 빔을 감자에 겨누면 성장 게이지가 오른다(완전 성숙까지 약 20초 분량).
- 성장 50%를 넘기면 두둑 둘레로 **감자알**이 하나씩 돋아나고(한 그루당 5개), 돋을 때마다 초록 입자 연출 + 상승하는 수확음이 난다.
- 한 그루가 감자알 5개를 모두 맺으면 성숙(항상 발광·보호). 살아남은 감자가 모두 성숙하면 **승리**, 모두 파괴되면 **패배**.
- 결과 화면에서 **[R]** 키로 언제든 다시 시작.

## 기술 메모

- 제스처 판별은 손가락 끝-뿌리 거리를 **손 전체 크기(손목~중지뿌리)로 나눈 비율**로 계산해, 손을 기울여도 안정적이다.
- 상태 전환은 몇 프레임 연속 유지될 때만 반영해(디바운스) 오탐을 줄인다.
- 5단계 분사 지점은 `TerraGun.getMuzzleWorldPosition()`으로 이미 얻을 수 있다.
- MediaPipe는 `@mediapipe/tasks-vision`의 `HandLandmarker`를 사용하며, wasm/모델은 CDN에서 로드한다(첫 실행 시 인터넷 필요).
