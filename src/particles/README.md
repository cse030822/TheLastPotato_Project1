# particles/ — 물·불꽃 파티클 효과 (5단계 1·2번 구현됨)

가산 혼합(Additive) `THREE.Points` 기반 파티클 풀.

**구현됨 (5단계 1·2번):**
- `ParticleField.ts` — 파티클 풀 + 소프트 라운드 포인트 셰이더 + CPU 시뮬레이션
- `JetController.ts` — 두 테라-건의 분사 제어
  - 오른손 FIRE → **형광 블루 물**: 시안~블루 물방울이 뭉쳐 뿜어지고, 화성 중력으로 아래로 휘는 포물선. 겹치면 더 밝고, 지나간 자리에 옅은 잔상.
  - 왼손 FIRE → **붉은 불꽃**: 중심 하양·노랑 / 바깥 주황·빨강, 중력 거의 없이 곧고 빠르게 + 좌우 미세 흔들림, 생성 직후 부풀었다 작아지며 소멸 후 빛무리.
  - FIRE가 유지되는 동안만 생성, 상태가 풀리면 즉시 멈춤.

**아직 미구현 (5단계 3~5번, 그 이후):**
- 감자/곤충 충돌 반응 (물→성장, 불꽃→소멸)
- 최대 파티클 수 화면 표시(내부 상한 링버퍼로 개수 자체는 제한됨)

> 분사 지점/방향은 `TerraGun.getMuzzleWorldPosition()` / `getMuzzleWorldDirection()`으로 얻는다.
